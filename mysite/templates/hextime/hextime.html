This clock is like a binary clock. This 24 hour clock has its values converted to binary values, but there's a twist.

The hours are converted to binary, but the minutes and seconds are treated differently. They are converted to binary and concatenated together, resulting in a 12-bit code like so:

"010011" + "011101" -> 010011011101 -> 4DD

This 12-bit code is then converted to hexadecimal. Hence, hextime.
